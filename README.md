# Monthly Investment Calculator (Python)

## Description
A simple Python program that calculates monthly investments using compound growth.
Users can define the investment duration, monthly contribution, and growth rate to
see how their investment grows over time.

## Features
- Monthly investment simulation
- Compound growth calculation
- Customizable duration, contribution, and growth rate
- Clear month-by-month output

## Usage
Run the script and enter the required values when prompted:

```bash
python investment_calculator.py
